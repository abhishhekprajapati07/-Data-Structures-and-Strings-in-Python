# -Data-Structures-and-Strings-in-Python

This repository contains simple Python programs demonstrating basic concepts like dictionaries and list slicing. It is designed for beginners to understand fundamental operations in Python.

---

## 📌 Tasks Included

### 🔹 Task 1: Dictionary of Student Marks
- Create a dictionary with student names as keys and marks as values.
- Take user input for a student name.
- Display the corresponding marks.
- Show a message if the student is not found.

### 🔹 Task 2: List Slicing
- Create a list of numbers from 1 to 10.
- Extract the first five elements.
- Reverse the extracted elements.
- Print both the original and modified lists.
